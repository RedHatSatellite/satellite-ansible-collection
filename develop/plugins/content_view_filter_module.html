redhat.satellite.content_view_filter module – Manage Content View Filters

Note

This module is part of the redhat.satellite collection (version 3.3.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install redhat.satellite.

To use it in a playbook, specify: redhat.satellite.content_view_filter.

New in version 1.0.0: of redhat.satellite

Synopsis

  • Create and manage content View filters

Examples

- name: Exclude csh
  redhat.satellite.content_view_filter:
    username: "admin"
    password: "changeme"
    server_url: "https://satellite.example.com"
    name: "package filter 1"
    organization: "Default Organization"
    content_view: Web Servers
    filter_type: "rpm"
    package_name: tcsh

- name: Include newer csh versions
  redhat.satellite.content_view_filter:
    username: "admin"
    password: "changeme"
    server_url: "https://satellite.example.com"
    name: "package filter 1"
    organization: "Default Organization"
    content_view: Web Servers
    filter_type: "rpm"
    package_name: tcsh
    min_version: 6.20.00
    inclusion: True
